# Technologies

Technology stack detection. These configurable fields tell you whether a company uses specific technologies you've defined, identified through their website or job descriptions.

## Fields

| Field | Description |
| --- | --- |
| Has technologies that match \* | Whether matching technology was identified via company website or job descriptions |
| List of technologies that match \* | List of technologies identified via company website or job descriptions |

> Fields marked with \* are configurable and depend on your dataset setup.
