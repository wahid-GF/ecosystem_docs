# Company Information

The basics about every company in your dataset. Name, domain, size, type, and other firmographic data that forms the foundation of any account record.

## Fields

| Field | Description |
| --- | --- |
| Primary domain | Main website domain |
| Company name | Company name |
| Company type | Company type (e.g. public, private, non-profit) |
| Employee range | Range of employees (e.g. 51-200) |
| IPO status | IPO status of the company |
| LinkedIn description | Company's "About Us" description |
| Year founded | Year the company was founded |
| LinkedIn followers | Count of the company's social media followers |
| Country | Company's headquarters country |
| Employee count | Total employee count based on individual employee data, not estimated range |
| Revenue range | Estimated company revenue range |

> **Shared fields**: Country also appears in [Location](location.md). Employee count also appears in [Team](team.md).
